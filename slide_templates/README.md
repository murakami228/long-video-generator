# Slide Templates

スライド動画生成で使用できるHTML/CSSテンプレート集です。
`projects/` にコピーして使用してください。

## テンプレート一覧

| No | 名前 | 特徴 | 用途 |
|---|---|---|---|
| **01** | **Grid Blue** | 基本・青系 | 3つのポイント、箇条書き、汎用 |
| **02** | **Grid Cyber** | ダーク・ネオン・Glitch | AI、技術、未来、ゲーミング |
| **03** | **Split Minimal** | 白基調・左右分割 | メリット/デメリット、対比 |
| **04** | **Step Flow** | 縦方向・矢印 | 手順、ロードマップ、成長 |
| **05** | **Code Dark** | VS Code風 | プログラミング解説、コード比較 |
| **06** | **Big Impact** | 超巨大文字・赤 | インパクト重視、数字の強調 |
| **07** | **Book Summary** | 明朝体・紙質感 | 書評、格言、落ち着いた解説 |
| **08** | **List Pop** | 黄色・丸文字 | ポップなランキング、日常、元気 |
| **09** | **Warning Alert** | 警告色・斜め線 | セキュリティ、注意喚起、危険 |
| **10** | **Split Dark** | 黒基調・左右分割 | ガジェット比較、ダークモード |

## 使い方

1. `projects/` 内に新しいプロジェクトフォルダを作成します。
2. 使いたいテンプレートの `.html` と `.css` をコピーします。
3. ファイル名を `slide_default.html`, `slide_default.css` にリネームします。
4. HTMLの中身を編集して、動画の内容に合わせます。
