## 予定される変更

### [NEW] プロジェクトフォルダ: `projects/logistics_2025/`

#### [NEW] `slide_*.html` / `slide_*.css`
各スライドのデザインをHTML/CSSで構築します。

- `input_06.json` (エンディング)

## 検証プラン

### 自動化スクリプトによる検証
1. `scripts/merge_scripts.ts` を実行し、`input.json` を生成。
2. `scripts/validate_input.ts` を実行し、構成とテキスト内容をチェック。
3. `scripts/generate_audio_batch.ts` を実行し、VoiceVoxで音声を生成。
4. `scripts/sync_audio.ts` を実行し、`data.json` を生成。

### 手動検証
- `npm run dev` 実行中のブラウザプレビュー（Remotion）で以下を確認：
    - スライドの切り替えがナレーションと同期しているか。
    - スライド3のステップアニメーションが正しく動作しているか。
    - フォントやレイアウトに崩れがないか。
