# ロング動画自動生成プロジェクト 実装計画

プロジェクトの目的は、既存のショート動画生成システム (`video-generator`) をベースに、YouTubeのロング動画（横長、長尺）を生成するための新しい環境 (`long-video-generator`) を構築することです。

## ユーザーレビューが必要な事項
- 現時点では特になし。初期構築後にプロトタイプを通して要件を詳細化します。

## 提案される変更

### 1. プロジェクト初期構築
#### [NEW] `long-video-generator/`
- 既存の `video-generator` から必要なファイルをコピー。
- 不要な成果物ディレクトリ (`out`, `completed_projects` 等) や依存関係 (`node_modules`) は除外。

### 2. 環境設定
#### [MODIFY] `package.json`
- プロジェクト名などを適切に変更。
- Scripts の調整（必要であれば）。

#### [MODIFY] `remotion.config.ts`
- 動画の解像度（1920x1080想定）、FPSなどの設定確認と調整。

### 3. アセットとソースコードの整理
#### [DELETE/MODIFY] `public/`
- ショート動画用の古いアセットを削除。
- 共通で使用できるアセット（フォント、BGMなど）は保持。

#### [MODIFY] `src/`
- ショート動画用のコンポーネントをベースにしつつ、横長レイアウト用に調整するための準備。

### 4. スクリプトの移行
#### [MODIFY] `scripts/`
- 音声生成やスクリプト生成のロジックがロング動画の構造（例: 導入〜本編〜まとめ）に対応できるか確認。
- 当面は既存スクリプトを流用し、動作確認を行う。

## 検証計画

### 自動テスト
- `npm install` が正常に完了すること。
- `npm run start` でRemotionプレビューが起動すること。

### 手動検証
- テスト用の `input.json` を作成し、一連の生成フロー（音声生成 -> 動画レンダリング）がエラーなく通ることを確認する。
