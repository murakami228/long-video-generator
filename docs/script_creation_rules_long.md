# ロング動画用台本作成ルール

## 1. コンセプト
- **「3選」の廃止**
  - 従来の「おすすめ〇〇3選」のような羅列型の形式は原則使用しない。
  - **「1つのテーマ」を深く掘り下げる**構成とする。
  - 視聴者が1つのトピックについて深い理解や納得感を得られることをゴールとする。

## 2. 構成の原則
1. **導入 (Hook)**
   - 視聴者の悩みや課題を明確にし、本動画で解決する「1つの答え」を提示する。
2. **徹底解説 (Deep Dive)**
   - 表面的な紹介ではなく、仕組み、背景、具体例、メリット・デメリットなどを多角的に解説する。
   - スライドは情報の羅列ではなく、**解説の補助（図解、フロー、比較）**として使用する。
3. **結論・アクション (Action)**
   - 結局どうすればよいか、視聴者が次に取るべき具体的なアクションを提示する。

## 3. スライド活用方針
- **情報の厳選**
  - スライドに文字を詰め込みすぎない。
  - 1スライドにつき1メッセージ（ワンメッセージ）を徹底する。
- **視覚化**
  - テキストよりも、フローチャート、比較表、強調文字、アイコンを活用して直感的に理解できるようにする。

## 4. 禁止事項
- 脈絡のない情報の並列（例：機能Aと機能Bと機能Cをただ並べるだけ）
- 浅く広いだけの解説
- 「3選」「5選」といった安易なナンバリングタイトルの使用
